## 🎨 Ciallo 画板：基于 EasyX 的 C++ 像素画板



### 📝 简介

**Ciallo 画板** 是一个使用 C++ 语言和 [EasyX 图形库](https://www.easyx.cn/) 开发的简易像素画编辑器。它提供了一个直观的界面和丰富的绘图功能，允许用户在画布上进行像素级别的创作，并支持保存/加载作品以及制作简单的逐帧动画。

------

### 💻 技术栈

- **编程语言：** C++
- **图形库：** EasyX (适用于 Windows 平台)
- **构建系统：** CMake (推荐使用 CLion 等 IDE)
- **操作系统：** Windows

------

### 🌟 主要功能

#### 🎨 基础绘图功能

- 自定义画布尺寸。
- 鼠标**左键涂色**，**右键擦除**像素。
- 支持 9 种预设颜色快速切换。

#### 💾 文件和操作

- **保存/加载画布：** 将当前作品保存到文件或从文件加载 (`.cav` 格式)。
- **清空画布：** 一键将画布恢复为白色。
- **撤销/重做：** 支持多步操作的撤销 (`Undo`) 和重做 (`Redo`)。
- **高级工具：** 油漆桶（填充相邻同色区域）、颜色吸管（吸取鼠标位置的颜色）。

#### 🎬 动画功能

- **保存动画帧：** 将当前画布作为一帧动画保存。
- **播放动画：** 循环加载并显示所有已保存的动画帧。
- **删除动画：** 清除所有已保存的动画帧文件。

#### 🎵 互动与多媒体

- 项目包含**背景音乐**和各种**操作音效**，增加了用户体验。
- 美观的启动界面和加载进度条。

------

### 🛠️ 编译与环境配置 (致贡献者/使用者)

本项目依赖于特定的库和 IDE 设置，请确保完成以下配置才能正常运行：

#### 1. 字体依赖 (必须)

本项目界面使用了自定义字体。请务必将项目根目录下的 **`请安装我.ttf`** (汉仪像素 11px U) 字体文件**安装**到你的电脑系统中。

#### 2. EasyX 库配置

请确保你的 IDE/编译器已正确安装和配置了 **EasyX 图形库**。本项目通过 `CMakeLists.txt` 链接了相关库文件 (`libeasyx.a`, `easyx`, `winmm`)。

#### 3. CLion 工作目录设置 (关键)

为了让程序能正确加载 `resources` 文件夹中的图片和音频，**必须**手动设置 CLion 的工作目录：

1. 打开 CLion 的 **"Edit Configurations..."**。
2. 在你的应用程序配置 (`Ciallo_Printer`) 中，将 **"Working directory"** 设置为项目的 **根目录** (即包含 `CMakeLists.txt` 和 `resources` 的目录)。
   - **请勿**将其设置为 `cmake-build-debug` 或 `src`。

------

### 🚀 启动流程

程序启动后，请按照以下步骤操作：

1. 首先，程序会提示你输入画布的**宽度和高度**。
2. 完成输入后，程序将显示一个启动界面。
3. 点击 **【开始】** 按钮进入画板界面，或点击 **【退出】** 按钮关闭程序。
