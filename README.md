# Ciallo 画板：基于 EasyX 的 C++ 像素画板

## 简介

**Ciallo 画板** 是一个使用 C++ 语言和 **EasyX 图形库** 开发的简易像素画编辑器。它提供了一个直观的界面和丰富的绘图功能，允许用户在画布上进行像素级别的创作，并支持保存/加载作品以及制作简单的逐帧动画。

## 技术栈

* **编程语言：** C++
* **图形库：** EasyX (适用于 Windows 平台)
* **操作系统：** Windows

## 主要功能

### 🎨 基础绘图功能
* 自定义画布尺寸。
* 鼠标左键涂色，右键擦除像素。
* 支持 9 种预设颜色快速切换。

### 💾 文件和操作
* **保存/加载画布：** 将当前作品保存到文件或从文件加载 (`.cav` 格式)。
* **清空画布：** 一键将画布恢复为白色。
* **撤销/重做：** 支持多步操作的撤销 (Undo) 和重做 (Redo)。
* **油漆桶：** 一次性填充相邻的同色区域。
* **颜色吸管**：吸取鼠标位置的颜色。

### 🎬 动画功能
* **保存动画帧：** 将当前画布作为一帧动画保存。
* **播放动画：** 循环加载并显示所有已保存的动画帧。
* **删除动画：** 清除所有已保存的动画帧文件。

### 🎵 互动与多媒体
* 项目包含背景音乐 和各种操作音效，增加了用户体验。
* 美观的启动界面和加载进度条。

## 使用说明

### 启动流程
1.  程序启动后，首先会提示你输入画布的宽度和高度。
2.  完成输入后，会出现一个启动界面。
3.  点击 **【开始】** 按钮进入画板界面，或点击 **【退出】** 按钮关闭程序。
